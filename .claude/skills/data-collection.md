# Skill: DataCollection

## Цель
Собрать датасет для ML-задачи из нескольких источников и вернуть унифицированный DataFrame.

## Когда спрашивать пользователя
- **Спроси задачу** — если не указана: "Какую ML-задачу решаем? Какие данные нужны?"
- **Спроси подтверждение источников** — предложи найденные источники, жди аппрув перед скачиванием
- **Спроси объём** — сколько записей скачать (по умолчанию 2000)
- НЕ спрашивай про технические детали (формат файлов, названия колонок)

## Порядок действий

1. **Понять задачу** — получи от пользователя описание ML-задачи (sentiment, классификация, etc.)

2. **Найти источники** — предложи 2+ источника:
   - Open dataset: поищи на HuggingFace по ключевым словам задачи
   - Scraping/API: предложи подходящий сайт или API
   - Жди подтверждения от пользователя

3. **Собрать данные** — используй скрипт:
   ```
   skills/data_collection/scripts/data_collection_agent.py
   ```
   Обнови `config.yaml` под выбранные источники перед запуском.

4. **Проверить результат** — убедись что DataFrame содержит колонки:
   `text`, `label`, `source`, `collected_at`

5. **Провести EDA** — запусти анализ:
   - Распределение классов
   - Длины текстов
   - Топ-20 слов
   - Сохрани графики в `reports/`

6. **Сообщить итог** — скажи пользователю:
   - Сколько записей собрано
   - Из каких источников
   - Есть ли дисбаланс классов
   - Путь к сохранённым данным: `data/raw/collected.parquet`

## Выходной формат
```python
pd.DataFrame с колонками: text, label, source, collected_at
Сохранено в: data/raw/collected.parquet
```

## Доступные скрипты
- `scripts/data_collection_agent.py` — основной агент (скиллы: scrape, fetch_api, load_dataset, merge)
- `config.yaml` — конфигурация источников (обнови перед запуском)

## Требования к результату
- Минимум 2 источника данных
- Все источники в едином формате
- Данные сохранены в `data/raw/`
